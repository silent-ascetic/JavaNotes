# 多态

>同一个行为具有多个不同表现形式或形态的能力。

Java中多态的具体体现
- 方法重载
- 方法重写（覆盖）
- 多态参数（新内容，重点、难点）


## 向上造型

> 向上造型

```java
class Pet{}
class Dog extends Pet{}
class Penguin extends Pet{}
Dog dog = new Dog();
Pet pet = new Dog() //父类的引用指向子类的对象--向上造型
```
简单来说就是父类引用可以指向子类对象。
**注意：**
- 子类引用不能指向父类对象。
- 引用调用的类方法只能是父类中的。
